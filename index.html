List of dynamic layouts

  1. One Row Layout - Height have 100% of browser window height.
  2. Two Columns Layout - Toggle show/hide sidebar menu. No transition effect. Sidebar will always have same height as section.
  3. Two Columns Layout - Transition - Same as above but with transition CSS effect.
  4. Two Columns Layout - Collapsible - Another version of collapsible sidebar. Left sidebar is moving to the right.
  5. Two Columns Layout - Flex - CSS Flexbox applied.
  6. Triplet - Flex - Top menu, a sidebar and main content area. CSS Flexbox used as layout system.

Flexbox Cheat Sheet

Short tutorial is here.
Browser compatibility is here. 
.flexcontainer {
  display: flex | inline-flex;
  flex-direction: row | row-reverse | column | column-reverse;
  flex-wrap: nowrap | wrap | wrap-reverse;
  flex-flow: ‘flex-direction’ || ‘flex-wrap’ //shorthand
  justify-content: flex-start | flex-end | center | space-between | space-around; //horizontal align of items
  align-items: flex-start | flex-end | center | baseline | stretch; //vertical align of items
  align-content: flex-start | flex-end | center | space-between | space-around | stretch; //align flex lines

}

.flexitem {
  order: Integer;
  flex-grow: Number; /* default 0 */
  flex-shrink: Number; /* default 1 */
  flex-basis: Length(px, em, %) | auto; /* default auto */
  flex: none | [ 'flex-grow' 'flex-shrink'? || 'flex-basis' ] //shorthand
  align-self: auto | flex-start | flex-end | center | baseline | stretch; //same as align-items but for specific item
}
