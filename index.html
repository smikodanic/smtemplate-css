List of dynamic layouts

  1. One Row Layout - Height have 100% of browser window height.
  2. Two Columns Layout - Toggle show/hide sidebar menu. No transition effect. Sidebar will always have same height as section.
  3. Two Columns Layout - Transition - Same as above but with transition CSS effect.
  4. Two Columns Layout - Collapsible - Another version of collapsible sidebar. Left sidebar is moving to the right.
  5. Two Columns Layout - Flex - CSS Flexbox applied.
  6. Triplet - Flex - Top menu, a sidebar and main content area. CSS Flexbox used as layout system.
